## Chrome MCP Usage

The Chrome MCP server is now configured and ready for browser testing as specified in Feature 006 (Design Sync Integration). Available capabilities:

- **Navigation**: Navigate to URLs, go back/forward, reload pages
- **Interaction**: Click elements, hover, fill forms, upload files
- **Testing**: Take screenshots, start/stop performance traces, handle dialogs
- **Inspection**: List pages, get network requests, evaluate JavaScript

### Example MCP Tool Call (via code)
```typescript
// Navigate to a page
await mcpClient.call('mcp_io_github_chr_navigate_page', {
  type: 'url',
  url: 'https://localhost:3000'
});

// Take screenshot
await mcpClient.call('mcp_io_github_chr_take_screenshot', {
  format: 'png',
  quality: 90
});
```
